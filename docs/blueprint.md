# **App Name**: Konk Media Archive

## Core Features:

- User Registration: Allow users to register and authenticate, with admin and regular user roles.
- Video Upload: Enable users to upload video fragments with titles and descriptions.
- Admin Approval: Implement a moderation queue where admins can approve or reject uploaded videos.
- Video Search: Allow users to search for videos by keywords found in titles or descriptions.
- Video Display and Download: Display approved videos for users to watch and download.
- Content Recommendations: Use AI to suggest videos based on viewing history, title and description analysis, acting as a tool for content discovery.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) inspired by the retro tech aesthetic of the YouTube channel's banner.
- Background color: Dark Navy (#1A237A), a desaturated, darkened shade of the primary hue for contrast.
- Accent color: Vibrant Purple (#BE29EC), an analogous color for highlights and call to action elements.
- Headline font: 'Space Grotesk' (sans-serif) for a techy feel.
- Body font: 'Inter' (sans-serif) as it's good for readability.
- Use simple, outline-style icons for clarity and a modern feel.
- Maintain a clean and intuitive layout, similar to yarn.co, with clear video previews and search functionality.
